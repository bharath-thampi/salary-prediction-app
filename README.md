# salary-prediction-app

This is a Machine Learning web application that predicts salary based on years of experience, interview score and the test score.

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML
- css
- Render (for deployment)

## Project Structure
salary-predictor/
│
├── app.py
├── model.pkl
├── requirements.txt
├── templates/
│   └── index.html
└── README.md

## How to Run

1. Clone the repository
git clone https://github.com/bharath-thampi/salary-prediction-app.git

2. Install dependencies
pip install -r requirements.txt

3. Run the application
python app.py

4. Open browser
[http://127.0.0.1:5000/](https://salary-prediction-app-1-uwd5.onrender.com/)

## Deployment
The project is deployed using Render.
